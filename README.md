# Shayboub - Specialty Coffee Website

A modern, performant website for Shayboub (شايبوب) - a specialty coffee & artisan sandwich café with locations in Cairo and Alexandria, Egypt.

## 🚀 Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Testing**: Vitest + Playwright
- **State Management**: TanStack Query

## ✨ Features

- Responsive design for all devices
- Fast image loading with lazy loading
- SEO optimized with JSON-LD structured data
- Accessible (WCAG compliant)
- Smooth animations
- Interactive menu with category tabs

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```
